## Run Locally

Clone the project

```bash
  git clone https://github.com/lukaswski/unicard_frontend
```

Go to the project directory

```bash
  cd unicard_frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
